# Form-Specific Tips: Business Writing

Apply these in addition to the general principles when the text is identified as business writing (emails, memos, reports, proposals, internal communications, etc.).

---

## What to look for and apply

**Replace jargon with plain language**
Work through the revised text word by word and replace every instance of corporate jargon. After making other edits, do a final pass specifically checking for the terms below — they commonly survive a first editing pass because they feel normal in business contexts. Every one of them should go.

| Jargon → Plain language |
|---|
| "revert when you have bandwidth" → "reply when you have time" |
| "we will experience a shortfall" → "we'll lose money" (or name the actual number) |
| "leverage our core competencies" → "use our strengths" |
| "circle back" → "follow up" |
| "touch base" → "talk," "meet," or "check in" |
| "going forward" → "from now on" or just delete it |
| "at this juncture" → "now" |
| "in terms of" → restructure the sentence |
| "synergize" / "synergy" → "work together" / "collaboration" |
| "deliverable" → "result," "output," or name the specific thing (e.g. "report," "design," "prototype") |
| "deliverables" → "work," "tasks," or name the specific things |
| "action item" → "task" |
| "deep dive" → "close look," "thorough review," or "detailed discussion" |
| "scalable" → be specific about what will scale and how |
| "strategic initiative" → name the actual thing being done (e.g. "the new pricing rollout," "the hiring plan") |
| "strategic initiatives" → name them, or use "plans," "projects," or "priorities" |
| "pain points" → "problems" |
| "move the needle" → be specific about what change is expected |
| "bandwidth" (meaning capacity/time) → "time," "capacity," or "availability" |
| "actioned" / "action this" → "done," "completed," "handle this," "do this" |
| "slippage" → "falling behind," "delays," or "missing the deadline" |
| "stakeholders" → name the people (e.g. "the team," "the client," "senior leadership") |

**Final check:** After editing, re-read the revised text once more looking only for jargon. It is easy to remove some terms while leaving others — particularly "deliverables," "strategic initiatives," and "stakeholders," which feel so embedded in workplace language that they can pass unnoticed.

**Remove euphemisms**
- Business writing often obscures bad news. Where a euphemism softens something that should be stated plainly, flag it.
- "Restructuring" → "layoffs" (where that is what is meant); "challenging quarter" → "a quarter in which we lost money."

**Write like a human, not a bureaucracy**
- Remove the passive voice that bureaucratic writing favours.
- Write to one reader, not to a generic audience. "You" and "we" are strong words.
- Contractions ("we'll," "don't," "it's") make business writing feel less cold.

**State the point early**
- Business writing should open with the main point, not build to it. The reader's time is limited.
- If the key ask, conclusion, or recommendation appears at the end, move it to the first paragraph.

**Avoid "throat-clearing" openings**
- Flag openings like "I am writing to you today to..." or "As per our previous discussion..." — these delay the real message. Cut them and start with substance.
